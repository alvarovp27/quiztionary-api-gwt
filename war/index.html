You ought to know...

This API have a token authentication system. To use our API, you first need a authorization token. For all the request, you need to specify in the headers request your current token in this way: "Authentication": "Your token" You can sign up by requesting as a POST HTTP message, including in the body the fields "name" and "password". Automatically, you will obtain your authentication token. This is the URI to which you have to send this request: http://www.quiztionary-api.appspot.com/api/users/new-user When you log in, you will get a new token, and all the ancient tokens will be invalidated.
